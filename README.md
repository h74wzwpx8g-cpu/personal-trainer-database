# Personal Trainer Database System

## Overview
A database system designed to manage personal trainers, clients, and training sessions efficiently.

## Features
- Store trainer and client information
- Manage training sessions and schedules
- Query data using SQL (JOIN, GROUP BY, HAVING)
- Data organization using relational schema

## Technologies
- SQL
- Database Design (ERD, Relational Schema)
